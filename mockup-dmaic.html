Guide SIPOC

Qu'est-ce que le SIPOC ?

Le SIPOC est un outil de cartographie de processus qui identifie les éléments clés : Suppliers (Fournisseurs), Inputs (Entrées), Process (Processus), Outputs (Sorties) et Customers (Clients).

Comment remplir ?

1. Commencez par le Process - listez les étapes principales

2. Identifiez les Outputs - que produit le processus ?

3. Définissez les Customers - qui reçoit les outputs ?

4. Listez les Inputs - de quoi avez-vous besoin ?

5. Identifiez les Suppliers - qui fournit les inputs ?

Conseil : Restez à un niveau macro (5-7 étapes max pour le processus). Le détail viendra dans les analyses suivantes.
